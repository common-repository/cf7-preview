== Installation ==

#### Thank you for your interest in `Preview Form`. 
### Minimum requirements.

* Wordpress 3.0+
* PHP 5.x.
* MySQL 5.x.

This section describes how to install the plugin and get it working. 
e.g.

1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
2. Activate the "Preview Form" plugin through the plugins menu in WordPress.
3. Go to edit form in 'Contact Form 7'.
4. Click `Preview` tab.

If any problem occurs, please read documents Wordpress.
